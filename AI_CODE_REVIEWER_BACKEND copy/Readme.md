
# 🚀 AI Coding Contest Platform

## 🎯 What is this?
A smart coding platform where programmers can compete in contests and get AI-powered code reviews!

Think of it like **LeetCode + ChatGPT combined!** 🤖💻

## ✨ Cool Features

### 🏆 Coding Contests
- Join programming competitions
- Solve challenging problems
- Compete with other coders
- Live leaderboard

### 🤖 AI Code Review
- Instant tips on your code
- Detailed analysis from AI
- Chat with AI about your code
- Learn to write better code

### 💻 Smart Editor
- Code in multiple languages (Python, Java, C++, JavaScript)
- Real-time execution with Judge0
- Save your solutions
- GitHub integration

## 🛠️ How it works?

### For Coders:
1. Sign up for an account
2. Join a contest
3. Solve problems in the editor
4. Get AI feedback instantly
5. Chat with AI for help
6. Climb the leaderboard 🏅

### For Contest Hosts:
1. Create coding challenges
2. Set up contests
3. Invite participants
4. Monitor submissions

## 🔧 Tech Stuff (For Developers)

### Microservices Architecture:
- **Auth Service** 👤 - User accounts & login
- **Contest Service** 🏆 - Competitions & problems
- **Editor Service** 💻 - Code editing & execution
- **AI Service** 🤖 - Code reviews & AI chat
- **MongoDB** 🗄️ - Database

### Built With:
- **Docker** 🐳 - Everything runs in containers
- **Node.js** ⚡ - Backend services
- **React** ⚛️ - Frontend (coming soon)
- **MongoDB** 🍃 - Database
- **Judge0** ⚖️ - Code execution

## 🚀 Quick Start

### Prerequisites:
- Docker installed on your machine
- That's it! 😊

### Run the project:
```bash
# Clone the project 
git clone https://github.com/MOHDDANISH007/AI-Code-Reviewer.git
cd Final_Year_Project

# Start everything with one command!
docker-compose up --build

# That's it! Your platform is running! 🎉
```

### Access the services:
- **Auth Service**: http://localhost:3001
- **Contest Service**: http://localhost:3003
- **Editor Service**: http://localhost:3002
- **AI Service**: http://localhost:3004## 
📁 Project Structure
```
coding-platform/
├── 🐳 docker-compose.yml    # Runs everything together
├── 📁 services/            # All backend services
│   ├── 👤 auth/           # User authentication
│   ├── 🏆 contest/        # Contests & problems
│   ├── 💻 editor/         # Code editor
│   └── 🤖 ai/            # AI magic!
└── 📁 database/           # Database setup
```
